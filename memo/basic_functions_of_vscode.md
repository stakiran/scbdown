# ● VS Code 拡張機能として実装できるものは:
基本機能

1. シンタックスハイライト — TextMate grammar で scbdownの各要素(見出し、リスト、リンク、画像、区切り線、コードブロック、引用等)を色分け
2. ファイル拡張子の関連付け — .scbdown や任意の拡張子を scbdown 言語として認識

エディタ支援

3. プレビュー — Webview パネルで scbdown をレンダリングして表示(リアルタイム更新)
4. アウトライン — サイドバーに見出し一覧を表示
5. リンクのホバー情報 — [xxx] にホバーするとリンク先を表示
6. リンク先へのジャンプ — 内部リンク [xxx] から xxx.md へ Go to Definition

発展的な機能

7. リンクの補完 — [ を入力したとき、既存ファイル名を候補表示
8. 折りたたみ — 見出しやコードブロック単位での折りたたみ
9. ファイル名バリデーション — 内部リンクのファイル名置換ルール(無効文字→_)の警告

どこから着手しますか？1+2 から始めるのが自然だと思います。
