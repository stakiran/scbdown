# 行塊（ぎょうかい）について
行塊について示すために、以前の会話ログを書いておく。

## 会話ログ
続いてジャンプ機能が欲しい。alt + up/down で前後の見出しにジャンプし、ctrl + uo/down で前後の行塊にジャンプします。

ここで行塊（ぎょうかい）とは、空行で区切られた行の塊を指します。たとえば以下には行塊が6つあります:

```smd
# 見出し1
あああ
 111
 111
  222
   333
いいい

ううう

## 見出し2
えええ
おおお

かかか

## 見出し3

ききき
 111
 111
  222
```

またジャンプはループしますので、先頭から末端に、また末端から先頭にもジャンプできます。
